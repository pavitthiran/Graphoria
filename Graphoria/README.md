# Graphoria Studio

A modern digital design studio website built with React, TypeScript, and Tailwind CSS.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open http://localhost:8080 in your browser

## Features

- Modern UI with React + TypeScript
- Tailwind CSS for styling
- Responsive design
- Smooth scrolling navigation
- Image carousel with auto-scroll

## Build

To create a production build:
```bash
npm run build
```

To preview the production build:
```bash
npm run preview
```
